# AI College Assistant

**AI College Assistant** is an AI-powered chatbot and voice assistant designed to help students quickly get answers to common college-related questions. Users can interact with it via a web interface, making it simple to access information about courses, faculty, events, and other college FAQs.  

---

## Features
- **Web-based Chat Interface**: Ask questions directly on a simple, intuitive webpage.  
- **FAQ Handling**: Retrieves answers from a pre-defined database of college-related questions.  
- **Easy Maintenance**: Questions and answers stored in a JSON file for easy updates.  

---

## Technologies Used
- **Python**: Core programming language  
- **Flask**: Web framework for backend and frontend integration  
- **HTML/CSS**: Frontend webpage design (`index.html`)  
- **JSON**: Stores FAQ data (`data/data1.json`)
